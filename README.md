# Binjyo

A simple screenshot tool inspired by SETUNA2.
Suitable for illustrators.

## Dependency

- .NET 4.6.1
- WPF

## Usage

- Ctrl+Alt+A or Ctrl+Win+A to screenshot

On a memo

- Double click to copy and close
- Ctrl+C and Ctrl+X is abled
- Esc to close
- D to size down
- F to size up
- R to reset size
- S to save png

click the button on top-right corner to switch mode:

- normal
- locked
- minimized
